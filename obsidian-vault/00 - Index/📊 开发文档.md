# 📚 博客开发文档

> 静墨博客系统的完整开发文档和使用指南

## 🚀 快速开始

### 一键启动

```
双击: start-all.bat
```

这会自动：
1. ✅ 检查 MongoDB
2. ✅ 安装依赖
3. ✅ 启动博客服务器
4. ✅ 启动 Obsidian 监控
5. ✅ 显示服务地址

| 服务 | 地址 | 说明 |

## 📝 Obsidian 工作流

### 发布文章

1. **在 Obsidian 中写笔记**
   - 位置：`obsidian-vault/02 - Notes/`
   - 使用树状结构分类

2. **编辑笔记**
   - 添加 Frontmatter 元数据
   - 完善内容

3. **移动到发布文件夹**
   - 将文件移动到 `01 - Blog/`
   - 自动同步到博客数据库

4. **查看效果**
   - 刷新博客首页
   - 点击查看文章详情

### Frontmatter 格式

```markdown
---
title: "文章标题"
category: "技术"
tags: ["JavaScript", "前端"]
published: true
featured: false
---

# 文章标题

正文内容...
```

### 字段说明

| 字段 | 必需 | 默认值 | 说明 |
|------|------|--------|------|
| title | ✅ | - | 文章标题 |
| category | ❌ | "随笔" | 分类 |
| tags | ❌ | [] | 标签数组 |
| published | ❌ | true | 是否发布 |
| featured | ❌ | false | 是否精选 |

## 🛠️ 技术栈

- **前端**：原生 HTML/CSS/JavaScript
- **后端**：Node.js, Express
- **数据库**：MongoDB
- **认证**：JWT
- **文件监控**：Chokidar
- **样式**：CSS3 + 动效

## 🔧 环境要求

- Node.js >= 14.x
- MongoDB >= 4.x
- npm 或 yarn

## 🐛 故障排除

### 问题 1：服务无法启动

```bash
# 检查 MongoDB
net start MongoDB

# 重新安装依赖
npm install
```

### 问题 2：无法登录

```bash
# 重置管理员密码
node reset-admin.js
```

### 问题 3：同步不生效

```bash
# 手动运行导入
node scripts/import-from-obsidian.js
```

### 问题 4：端口被占用

```bash
# 查看端口占用
netstat -ano | findstr :3000

# 结束进程
taskkill /PID <ID> /F
```
