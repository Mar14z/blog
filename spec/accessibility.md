# 可访问性与 SEO 规范

> 静墨博客（JingMo Blog）可访问性标准与 SEO 优化规范，所有页面必须遵循。

---

## 1. 可访问性（Accessibility）

### 语义化标签

必须使用以下语义化标签替代无语义的 `div`：

| 场景 | 标签 |
|------|------|
| 导航区域 | `<nav>` |
| 主内容区 | `<main>` |
| 文章内容 | `<article>` |
| 内容分区 | `<section>` |
| 侧边栏/补充信息 | `<aside>` |
| 页脚区域 | `<footer>` |

### 图片可访问性

- 所有 `<img>` 必须添加 `alt` 属性
- 装饰性图片使用 `alt=""`
- 有意义的图片使用描述性 `alt` 文本

### 表单可访问性

- 表单元素添加 `aria-label` 属性
- 搜索输入框：`aria-label="搜索文章"`

### 导航可访问性

- 导航添加 `role="navigation"` 和 `aria-label="主导航"`

### 链接安全

- 外部链接添加 `rel="noopener noreferrer"`

### 键盘导航支持

| 按键 | 功能 |
|------|------|
| `Tab` | 在可交互元素间切换焦点 |
| `ESC` | 关闭弹窗/模态框 |
| `←` `→` | 相册翻页 |

### 对比度

- 文字与背景对比度必须符合 **WCAG 2.1 AA** 标准
- 普通文本：对比度 ≥ 4.5:1
- 大文本（18px+ 或 14px+ 粗体）：对比度 ≥ 3:1

## 3. SEO 优化

### 必需 Meta 标签

每个页面必须包含以下 meta 标签：

```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="页面描述">
<title>页面标题</title>
```

### Title 格式

| 页面类型 | 格式 | 示例 |
|----------|------|------|
| 子页面 | `页面名 - 静墨` | 文章列表 - 静墨 |
| 首页 | `静墨 - 描述` | 静墨 - 极简博客 |

### 语义化标签与 SEO

- 文章页使用 `<article>` 标签
- 导航使用 `<nav>` 标签
- 页脚使用 `<footer>` 标签
