# 静墨博客 - 快速使用指南

## 一键启动

**双击 `start-all.bat`**

自动完成：
- ✅ MongoDB 检查
- ✅ 依赖安装
- ✅ 启动博客服务器
- ✅ 启动 Obsidian 监控

---

## 访问地址

- 博客：http://localhost:3000
- 管理后台：http://localhost:3000/admin
- 登录账户：admin / admin123

---

## 发布文章流程

### 1. 写笔记
打开 `obsidian-vault` 文件夹，进入 `02 - Notes` 选择合适的分类写笔记。

### 2. 添加 Frontmatter
```markdown
---
title: "文章标题"
category: "技术"
tags: ["JavaScript"]
published: true
---

# 文章标题

正文内容...
```

### 3. 发布
将笔记**移动**到 `01 - Blog` 文件夹。

### 4. 查看
等待 2 秒自动同步，刷新博客查看。

## Obsidian 目录结构

```
obsidian-vault/
├── 00 - Index/      # 索引和文档
├── 01 - Blog/       # 发布文章（自动同步）
└── 02 - Notes/      # 个人笔记
    ├── 00 - Inbox/  # 收集箱
    ├── 01 - Learning/ # 学习笔记
    ├── 02 - Projects/ # 项目笔记
    ├── 03 - Areas/  # 领域研究
    └── 04 - Resources/ # 资源收藏
```

## 常见问题

| 问题 | 解决 |
|------|------|
| 无法启动 | `net start MongoDB` |
| 无法登录 | `node reset-admin.js` |
| 同步失效 | 重启 start-all.bat |
